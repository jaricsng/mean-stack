# mean-stack
mean stack sample based on https://thinkster.io/tutorials/mean-stack : added menus, bootstraps

the mean stacks uses the following
- MongoDB
- Express
- AngularJS 1
- Bootstraps
- jwt
- Passport
- ngMaterial
